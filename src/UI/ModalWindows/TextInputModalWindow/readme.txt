Это модальное окно предназначено для ввода текстовой информации. Имеет поле для ввода информации
и кнопка для подтверждения ввода. Свойства всех компонентов можно менять и ставить им текстуры.

Для создания в конструктор нужно передать координаты верхнего левого угла, ширину и высоту, пояснение,
а также windowComponent - аналог window в браузере, нужен для работы с ивентами нажатия на кнопку.
На все компоненты окна можно повесить listener для обработки событий. Так как все рисуется на канвасе,
то нужны кастомные средства для обработки событий. Для этого на канвас вешаются обработчики нужных
ивентов. Вся работа этих обработчико заключается в том, что бы отправлять ивенты в EventQueue.
В методе update эти ивенты дастаются и передаются объекту класса UI, который отвественен за их обработку.

Это окно можно использовать, как, например, для ввода имени или ссылки на изображение
(хотя это не очень удобно, т.к. возможность копирования пока не реализована =)).

В примере реализованы две этих возможности. Его можно запустить start-example-modal
