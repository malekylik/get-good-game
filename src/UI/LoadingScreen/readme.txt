LoadingScreen представляет из себя: черный прямоугольник, который закрывает, то что еще не загрузилось;
поверх рисуется еще один прямоугольник, который меньше и он обтягивается текстурой, служащий задником для полосы прогресса;
сама полоса прогресса;

В конструктор передаются координаты верхнего левого угла черного прямоугольник, его ширина и высота, а также текстура.
У объекта LoadingScreen можно получить объект полосы прогресса, у которой есть метод для установки значение в процентном соотношение.
Но можно установить и любой другой текст. Можно менять свой любого компонента LoadingScreen такие как: размер, различные цвета, 
поставить тектуру на задний фон и другие.

Для использование нужно сначала подгрузить текстуру, а потом создать LoadingScreen. Для того, чтобы LoadingScreen ловит пользовательские
ивент, нужно его добавить в канвас как ui компонента, иначе как сцену.

Основное применение - это для отображение загрузки, но можно использовать и чего-нибудь другого, например, отображения прогресса
при прослушивание музыки.

Объедененые выше варианты объеденены в один пример. В примере реализовано создание LoadingScreen, для отображения загрузки музики,
что может быть не заметно, так как файл небольшой и загрузка происходит быстро. А затем LoadingScreen используется для отображения
прогресса при прослушивание музыки. LoadingScreen после загрузки не удалятся и не создается новый, а используется для дальнейшей цели,
дабы не делать лишние действия. 
Пример можно посмотреть, запустив скрип: start-example-loading-screen
